- component is comprized into three elements
a. design
b. style
c. logic
 * design is defined is useing html
 * styls are defined with css
 * logic is defined with jsx or tsx
 * technically component is  a javascript 
a)class
b)function
*  classes are supoorted by react but not recommended in modern code 
* react 19x version always recomended to design component with function

3. name must start  with  uppercase latter
